Quick start:
1) git clone https://github.com/igor-sergeevich-po/game-store.git
2) git checkout -b dev
3) git branch --set-upstream-to=origin/dev
4) git pull
5) npm i
6) npm start
7) use

[Use this app](https://igor-sergeevich-po.github.io/game-store/)
